##  Project Architecture
The project follows a standard 3-tier data pipeline:
1. **Extraction (SQL):** Raw data is structured into relational schemas.
2. **Transformation (Python):** Advanced algorithmic analysis (RFM & Apriori).
3. **Visualization (Power BI):** Interactive reporting for business stakeholders.
